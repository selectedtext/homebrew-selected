# Selectedtext Selected

## How do I install these formulae?

`brew install selectedtext/selected/<formula>`

Or `brew tap selectedtext/selected` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "selectedtext/selected"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).

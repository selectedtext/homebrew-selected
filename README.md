# Selectedtext Selected

## How do I install these formulae?

`brew install selectedtext/selected/selected`

Or `brew tap selectedtext/selected` and then `brew install selected`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "selectedtext/selected"
brew "selected"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
